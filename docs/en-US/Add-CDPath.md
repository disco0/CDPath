# Add-CDPath

## SYNOPSIS
Adds one or more paths to your cdpath

## SYNTAX

```
Add-CDPath [[-Path] <String[]>] [-Prepend] [<CommonParameters>]
```

## DESCRIPTION
This is useful if you have modified your path file with an external editor
